Aprender HTML desde Cero

HTML es el lenguaje de marcado estándar para crear páginas web. Define la estructura de la información mediante etiquetas.

Conceptos Fundamentales

  • Etiquetas: Se escriben entre ángulos como <p>.
  • Atributos: Dan información extra, como href o src.
  • Anidamiento: Unas etiquetas van dentro de otras.

Pasos para crear un documento

  1. Declarar el tipo de documento con <!DOCTYPE html>.
  2. Crear la etiqueta raíz <html>.
  3. Añadir el <head> para la configuración invisible.
  4. Escribir el contenido visible dentro del <body>.
