Overflow y Display (HTML)

Son dos propiedades fundamentales de CSS utilizadas para controlar el diseño (layout) y el comportamiento visual de los elementos en una página web, el display (Visualización) determina el tipo de caja que genera un elemento y cómo se comporta con respecto a otros elementos en el diseño.
